- [ ] LCP (Largest Contentful Paint) < 2.5s

- [ ] Время загрузки < 3 секунд
- [ ] Размер страницы < 1 MB (без изображений)
- [ ] Количество запросов < 30

## 📝 Контент стратегия

Основные фразы для ранжирования:

1. **High Priority (Объёмные)**
   - приложение маляра
   - окраска автомобилей

   - вязкость краски

3. **Long Tail (Специфические)**
