## 🛠️ Техническая поддержка

### Если возникли проблемы

**Ошибка 500:**
1. Переименуй .htaccess в .htaccess_backup
2. Обратись в поддержку хостинга
3. Попроси включить mod_rewrite

**Яндекс.Метрика не работает:**
1. Проверь ID счётчика
2. Добавь домен в настройки
3. Очисти кэш браузера

**robots.txt не открывается:**
1. Проверь загрузку файла
2. Убедись в корректности прав (chmod 644)
3. Проверь в корне сайта (не в подпапке)
